## 🚀 使用说明

### 本地运行

```bash
cd website

# Python
python3 -m http.server 8000

# Node.js
npx serve .

# PHP
php -S localhost:8000
```

浏览器访问：**http://localhost:8000**

### 键盘快捷键

| 快捷键 | 功能 |
|--------|------|
| 空格键 | 播放/暂停 |
| ← 方向键 | 后退5秒 |
| → 方向键 | 前进5秒 |
| M | 静音/取消静音 |
| F | 全屏/退出全屏 |
