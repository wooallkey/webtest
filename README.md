# VideoHub - 短视频平台

一个功能完整的短视频网站，参考抖音Web端设计风格。

## 功能特性

### 🎬 视频播放
- 主视频播放器，支持播放/暂停
- 进度条拖动跳转
- 音量控制（静音/取消静音）
- 全屏播放
- 自动循环播放

### 👥 用户交互
- **点赞** - 点击心形按钮，支持动画效果
- **评论** - 查看和发布评论
- **分享** - 分享到微信、微博、QQ等平台
- **收藏** - 收藏喜欢的视频
- **关注** - 关注喜欢的创作者

### 🔍 内容浏览
- 分类标签筛选（全部、旅行、美食、美妆、音乐、游戏、科技）
- 搜索功能
- 侧边导航栏（精选、推荐、关注、我的）
- 推荐视频列表

### 📤 内容创作
- 视频上传功能
- 填写标题和描述
- 拖拽上传支持

### ⌨️ 快捷键支持
- **空格键** - 播放/暂停
- **←** - 后退5秒
- **→** - 前进5秒
- **M** - 静音/取消静音
- **F** - 全屏/退出全屏

## 设计风格

- **深色主题** - 黑色/深灰色背景，减少眼睛疲劳
- **红色系主色调** - 活力、时尚
- **圆角设计** - 卡片、按钮使用圆角
- **毛玻璃效果** - 视频信息覆盖层渐变
- **响应式布局** - 支持桌面端和移动端

## 运行方式

```bash
cd website
python3 -m http.server 8000
# 或
npx serve .
```

然后在浏览器中访问: http://localhost:8000

## 技术栈

- 原生 HTML5
- 原生 CSS3 (CSS Variables, Flexbox, Grid)
- 原生 JavaScript (ES6+)
- 无外部依赖
