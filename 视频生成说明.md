# 视频素材生成说明

## AI视频生成限制说明

### 问题描述

火山方舟 Agent Plan API 目前**不支持视频生成模型**。

### 错误信息

```
"error": {
  "code": "UnsupportedModel",
  "message": "The requested model does not support the agent plan feature."
}
```

### 技术原因

1. **生图 Skill 正常工作**：
   - ✅ `byted-ark-seedream-skill` - 图片生成正常
   - ✅ API 端点：`/api/plan/v3/images/generations`

2. **生视频 Skill 受限**：
   - ❌ `byted-ark-seedance-skill` - Agent Plan API 不支持视频生成模型
   - ❌ 视频模型 `doubao-seedance-2.0` 未接入 Agent Plan 端点
   - ❌ 所有视频模型（1.5 pro / 2.0 / 2.0 mini / 2.0 fast）均返回 `UnsupportedModel` 错误

## 建议解决方案（如需真正AI视频）

### 方案1：使用非Agent Plan端点

火山方舟视频生成需要使用独立的API端点，需要：
1. 单独申请视频API权限
2. 使用 `/v1/videos/generations` 端点（非 `/api/plan/v3`）
3. 可能需要单独的AK/SK认证

### 方案2：控制台手动生成

1. 登录火山方舟控制台
2. 进入豆包 Seedance 视频生成
3. 手动生成后下载到项目目录

### 方案3：等待Agent Plan更新

后续火山方舟可能会更新Agent Plan API支持视频生成模型。
